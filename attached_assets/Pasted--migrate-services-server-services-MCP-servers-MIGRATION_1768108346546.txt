## Routes Integration

หลังจาก copy services แล้ว:

1. อัพเดท server/content-mcp/routes/index.ts:
   - Import services ทั้งหมดจาก ../services/
   - สร้าง routes สำหรับแต่ละ service
   - ใช้ async/await และ error handling

2. อัพเดท server/integration-mcp/routes/index.ts:
   - Import services ทั้งหมดจาก ../services/
   - สร้าง routes สำหรับแต่ละ service
   - ใช้ async/await และ error handling

3. ตรวจสอบ imports:
   - แก้ไข import paths ให้ถูกต้อง
   - เช็คว่าไม่มี dependency ที่หาย
   - ตรวจสอบ environment variables

## Testing

ทดสอบทุก endpoint ผ่าน Gateway:
- Content MCP: http://localhost:3000/api/content/*
- Integration MCP: http://localhost:3000/api/integration/*
